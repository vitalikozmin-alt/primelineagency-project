Вопросы и ответы

Почему YouTube видео не работает локально (при открытии файла из папки)?
Браузер блокирует iframe с внешних доменов при загрузке через file:// протокол — это политика безопасности. Это нормально и ожидаемо. Решения: (1) открывать через локальный сервер (npx serve .), (2) залить на хостинг.

Будет ли YouTube работать на хостинге?
Да, будет работать. Как только сайт загружен на реальный веб-сервер с доменом — embed через youtube-nocookie.com работает корректно. Кнопка play показывает превью, клик загружает iframe с autoplay.

Что если автор видео запретил embed?
Тогда iframe покажет ошибку. Решение: открыть настройки видео на YouTube → Правка → снять галочку "Запретить встраивание". Или использовать ссылку на YouTube напрямую вместо embed.

Правило версионности с этого момента:
Перед каждым крупным изменением — копировать текущий index.html под именем index-vN-описание.html. Актуальный файл всегда = index.html.
